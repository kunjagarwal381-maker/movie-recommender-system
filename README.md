# 🎬 Movie Recommender System

A content-based movie recommendation system built with Python and Streamlit.

## Features
- Recommends 5 similar movies
- Fetches real movie posters via TMDB API
- Clean and stylish UI

## Tech Stack
- Python
- Pandas & Scikit-learn
- Streamlit
- TMDB API
